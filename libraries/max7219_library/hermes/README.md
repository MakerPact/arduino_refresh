# Hermes Analysis Tools

This folder contains all Hermes-specific files for the Max7219 improvement project:

## Subfolders
- `analysis/` - Scripts for analyzing ghosting issues
- `testing/` - Automated test suites
- `documentation/` - Project-specific documentation
- `tools/` - Custom development tools

## Purpose
Keep all Hermes-generated content organized and separate from the main library code.
