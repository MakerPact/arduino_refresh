# Max7219 Library - Ghosting Fix Project

## Overview
This folder contains the Max7219 LED matrix driver library with improvements to fix ghosting effects (GitHub Issue #56).

## Structure
- `src/` - Source code (original + improvements)
- `examples/` - Usage examples
- `hermes/` - Hermes-specific analysis and testing tools
- `docs/` - Documentation
- `original/` - Backup of original library
